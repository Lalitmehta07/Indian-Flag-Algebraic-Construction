I. Project Methodology

This project was executed as a formal challenge to recreate the National Flag of India at a 1:1 scale using 100% algebraic equations. Following the official construction sheet, the flag width is set to 900 units, resulting in a 3:2 aspect ratio. Every horizontal band and every detail of the Chakra was calculated using strictly coordinate geometry.

III. The Rendering Paradox: Infinite Slopes

A significant technical discovery during this project was the failure of standard inequalities to render solid regions at vertical angles ($90^\circ$ and $270^\circ$). Mathematically, the slope approaches infinity, causing modern graphing engines to leave these regions uncolored.

Figure 2: Analysis of rendering artifacts caused by infinite slopes in the vertical domain.

Scientific Note: While the underlying logic remained strictly algebraic throughout the process, the polygon function was utilized for final presentation to bypass these software-specific pixel-sampling limitations.

IV. Conclusion

By combining engineering standards with vector geometry, this 1:1 scale model serves as a resolution-independent mathematical representation of the Indian National Flag.
